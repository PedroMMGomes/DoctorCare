Landing page responsiva para serviço de saúde, com seções de apresentação, slider de depoimentos e chamadas para contato. Projeto front-end estático (sem build).

## 🚀 Como rodar

```bash
cd src
npx serve .      # ou abra src/index.html no navegador
```

## 📁 Estrutura

```
src/
├─ index.html        # marcação da landing
├─ style.css         # estilos
├─ main.js           # interações (slider, menu)
└─ assets/           # imagens
```
